## ⚙️ Running Locally

To run this portfolio website locally on your machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Nikita-Gupta-19/My-Portfolio.git
   cd My-Portfolio
   ```

2. **Open the `index.html` file in your preferred web browser:**
   - Double-click `index.html` in your file explorer, OR
   - Run a simple local server if you want to avoid CORS limitations for advanced scripts:
     ```bash
     # Using Python 3
     python3 -m http.server 8000
     ```
     Then open `http://localhost:8000` in your browser.
